Class: Home

Home()

Class of the main component which is the main page the user sees before logging in, it contains the 3 component that form the homepage: the navbar, the main interface and the footer. The instance of this component is created and loaded as soon as the user opens Spotify's website.

A built in function that check if the component is on the screen, in other words makes sure we are on the home page to prevent CSS problems that causes the homepage's styling to be visible in other pages with different components
